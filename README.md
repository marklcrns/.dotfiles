# All my WSL and Linux dotfiles and configurations

Clone repo

```bash
git clone https://github.com/marklcrns/.dotfiles ~/.dotfiles
```

## Distribute Dotfiles

```bash
cd ~/Projects/.dotfile
./bin/tools/dotfiles/dotdist -VD -r .dotfilesrc . $HOME
```

## Packages Installation

```bash
cd ~/Projects/.dotfile
./scripts/install/install.sh
```
